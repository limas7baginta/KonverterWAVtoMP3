LimaskOnverter and WAVEform by Limas Baginta, Computer Engineering University of Indonesia.
Created 03/13/2017

Berikut cara menjalankan software LimaskOnverter

1. Buka folder Workshop 1

2. Buka Folder Software

3. Selanjutnya, masuk pada folder Software_LimaskOnverter lalu Klik LimaskOnverter.exe

4. Lalu muncul GUI untuk melakukan konversi dan klik pada “Browse WAV”

5. Selanjutnya memilih file (.wav) yang akan dikompresi menjadi file (.mp3) lalu klik Open. Pada
kesempatan ini, penulis memilih file Riot MP3.wav

6. Selanjutnya memilih VBR sesuai keinginan untuk dikompresi menjadi file (.mp3). Untuk VBR
q=4, output file (.mp3) berada sesuai dengan lokasi file (.wav) sedangkan untuk VBR q=1,
output file (.mp3) akan berada pada desktop secara default

7. Lalu tunggu hingga kompresi selesai dilakukan

8. Akhirnya, file (.mp3) akan muncul pada direktori yang sama dengan file (.wav) untuk VBR q=4,
sedangkan untuk VBR q=1, output file (.mp3) akan berada pada desktop secara default.
